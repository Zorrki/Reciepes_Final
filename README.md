# Проект “Сайт рецептов” на Django

## Краткое описание задания

Используя фреймворк Django, создайте сайт, на котором пользователи смогут добавлять свои рецепты блюд и просматривать рецепты других пользователей.

Готовый проект необходимо сдать в виде:
- ссылки на рабочий сайт в сети интернет;
- репозитория с исходным кодом проекта.

---

## Подробное описание задания

1. **Создайте проект Django и приложение(я) для сайта рецептов.**

2. **Модели**

   Для работы с пользователями используйте встроенного в Django User.

   Необходимые модели:

   - **Рецепты**
     - Название
     - Описание
     - Шаги приготовления
     - Время приготовления
     - Изображение
     - Автор
     - *другие поля на ваш выбор, например, ингредиенты и т.п.*
   
   - **Категории рецептов** (опционально)
     - Название
     - *другие поля на ваш выбор*

   - **Связующая таблица для связи Рецептов и Категорий** (опционально)
     - *обязательные для связи поля*
     - *другие поля на ваш выбор*

3. **Шаблоны**

   Подготовьте базовый шаблон проекта и следующие дочерние шаблоны:
   - Главная с 5 случайными рецептами (кратко)
   - Страница с одним подробным рецептом
   - Страницы регистрации, авторизации и выхода пользователя
   - Страница добавления/редактирования рецепта
   - *другие шаблоны на ваш выбор*

4. **Формы**

   - Создайте формы для ввода и редактирования информации (рецептов).
   - Интегрируйте их в шаблоны.

5. **Представления**

   - Создайте представления, которые охватывают все части проекта: модели, формы, шаблоны.

6. **Маршруты**

   - Подключите маршруты.
   - Убедитесь в работоспособности представлений и связанных с ними моделей, форм и шаблонов.

7. **Облачный сервер и наполнение**

   - Разверните рабочий проект на сервере.
   - Наполните базу данных как минимум пятью рецептами.

8. **Взаимодействие с одногруппниками** (если применимо)

   - Обменяйтесь ссылками с одногруппниками.
   - Заполните рецептами их сайт, а они — ваш.
